# 16. Installation aus Binärpaketen

PostgreSQL ist heute für die meisten verbreiteten Betriebssysteme in Form von Binärpaketen verfügbar. Wenn solche Pakete angeboten werden, ist dies für Benutzer des Systems der empfohlene Weg, PostgreSQL zu installieren. Das Bauen aus dem Quellcode, siehe [Kapitel 17](17_Installation_aus_dem_Quellcode.md), wird nur für Personen empfohlen, die PostgreSQL selbst oder Erweiterungen entwickeln.

Eine aktuelle Liste der Plattformen, für die Binärpakete bereitgestellt werden, finden Sie im Downloadbereich der PostgreSQL-Website unter <https://www.postgresql.org/download/>. Folgen Sie dort den Anweisungen für die jeweilige Plattform.
