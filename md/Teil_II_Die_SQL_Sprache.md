# Teil II. Die SQL-Sprache

Dieser Teil beschreibt die SQL-Sprache, wie sie von PostgreSQL implementiert wird. Er behandelt Syntax, Datendefinition, Datenmanipulation, Abfragen, Datentypen, Funktionen, Operatoren, Indizes, Volltextsuche, Nebenläufigkeit und Performance-Themen.

Wer SQL noch nicht kennt, sollte zuerst Teil I lesen. Eine knappe formale Übersicht der einzelnen SQL-Befehle steht in Teil VI.
